# AGENTS

This repo is a template for a .NET API backed by MongoDB and intended to be used via the Dev Container.

- Primary compose file: `.devcontainer/docker/docker-compose.backend-mongo.yml`
- Keep changes scoped to the template or devcontainer wiring unless asked to modify `src/`
- If you add new tooling or workflow steps, update `README.md` accordingly
- No automated tests are configured; ask before adding or running test commands
